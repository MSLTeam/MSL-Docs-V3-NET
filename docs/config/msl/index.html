配置Frp自动启动

  1. 获取需要自启动的Frp的ID

    进入 映射 选项卡，如图中所示 中括号 内的数字即为Frp隧道的ID，请记下来。

  2. 配置自启动

    在 MSL设置 ，自动化选项 中填入Frp隧道的ID。

    如果只需要自启动一个隧道，可以 直接填入此隧道ID。

    若需要自启动 多条隧道，请填入多个ID，并使用半角（英文）逗号,分割。如：123,321,

    填写完成后打开开关即可。
